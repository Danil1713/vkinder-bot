# 1. Описание проекта
VKinder Bot — это чат-бот для социальной сети ВКонтакте,
предназначенный для поиска знакомств. 
Бот анализирует профиль пользователя, определяет его пол, возраст и город, 
после чего подбирает кандидатов противоположного пола с похожими параметрами.
Для каждого кандидата бот показывает топ-3 фотографии с наибольшим количеством лайков.

# 2. Системные требования
- Python 3.8 или выше
- PostgreSQL 12 или выше
- Доступ в интернет для работы с VK API

# 3. Установка

### 3.1. Клонирование репозитория
git clone https://github.com/Danil1713/vkinder-bot.git cd vkinder-bot

### 3.2. Установка зависимостей
pip install -r requirements.txt

### 3.3. Настройка базы данных PostgreSQL
- Запустите PostgreSQL:
sudo service postgresql start # для Linux
или через pg_ctl # для Windows
- Создайте базу данных:
CREATE DATABASE VKinder.

# 4. Настройка (переменные окружения)
Все настройки хранятся в файле .env в корневой директории проекта. 
Создайте его на основе примера: .env.example

# 5. Запуск бота
python main.py

### При успешном запуске в консоли появится сообщение:

Подключение к БД установлено

Запуск бота VKinder...

# 6. Использование бота

Бот работает через личные сообщения ВКонтакте. 
Найдите ваше сообщество в поиске ВКонтакте и напишите ему.

### 6.1. Команды бота

### Основные команды

| Команда                         | Действие                                                                                                            |
|---------------------------------|---------------------------------------------------------------------------------------------------------------------|
| `Привет`, `start`, `начать`     | Инициализация сессии. Бот приветствует пользователя и отображает его данные (имя, возраст, город).                  |
| `Начать поиск`                  | Запуск поиска кандидатов. Бот находит до 20 пользователей противоположного пола в вашем городе с близким возрастом. |
| `Дальше`                        | Переход к следующему кандидату из найденного списка.                                                                |
| `В избранное`                   | Добавление текущего кандидата в список избранных.                                                                   |
| `В черный список`               | Добавление текущего кандидата в чёрный список (больше не будет показываться).                                       |
| `Показать избранное`            | Отображение списка всех сохранённых избранных кандидатов.                                                           |
| `Показать черный список`        | Отображение списка всех кандидатов,внесенных в  черный список.                                                      |

### 6.2. Интерфейс
При просмотре кандидата бот отправляет сообщение с:
- Именем и фамилией
- Ссылкой на профиль ВКонтакте
- Возрастом (если указан в профиле)
- Тремя лучшими фотографиями (по количеству лайков)

Под каждым сообщением расположены кнопки для управления:
- [Дальше]
- [В избранное] [В черный список]
- [Показать избранное] [Показать черный список]

# 7. Структура базы данных

| Таблица        | Назначение                                             |
|----------------|--------------------------------------------------------|
| `users`        | Хранение данных пользователей бота                     |
| `favorites`    | Избранные кандидаты (с фотографиями)                   |
| `blacklist`    | Чёрный список пользователей                            |
| `viewed_users` | Просмотренные кандидаты (чтобы не показывать повторно) |
| `photo_likes`  | (Дополнительно) информация о лайках фотографий         |
| `interests`    | (Дополнительно) интересы пользователей                 |


# 8. Дополнительная информация
- Возрастной диапазон для поиска: возраст пользователя ±5 лет (жёстко задано в коде);
- Статус: в активном поиске;
- Количество кандидатов за один поиск: до 20 человек;
- Фотографии берутся из профиля и системных альбомов пользователя.
